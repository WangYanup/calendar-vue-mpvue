# calendar-vue-mpvue

> vue 和  mpvue 通用的移动设备组件

文件路径： src/components/calendar.vue

## 功能
``` bash
1.支持农历和节假日（农历1900-2100）
2.支持左右滑动切换，和点击箭头切换
3.点击‘重置’按钮，回到当前日期
4.点击‘取消’按钮和灰色背景隐藏组件
5.支持特殊日期标记圆点
6.假期显示，需要在plugs/holida.data.js手动添加数据
7.支持日期范围选择
```

``` bash

npm install

# serve with hot reload at localhost:8080
npm run dev
```
